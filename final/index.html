URL

liammailley.com

Triggers

  • Hover
  • Click/Tap

Rules

  • On hover, Cursor changes to pointer
  • If a user hovers over one of the pictures in the accordion, that individual picture will expand and slightly zoom in
  • This will also cause a text overlay to slide up on the bottom of the picture, which gives a short summary of the section or project
  • Upon clicking on one of these pictures, a modal will appear detailing the project
  • Every time the user hovers over a new picture, a whoosh sound effect will play
